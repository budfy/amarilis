Класс Описание
.flex Добавьте класс к тегу, чтобы применить компонент Flex, это сразу же создаст гибкий контейнер. По умолчанию, все элементы Flex будут выровнены по левому краю и будут иметь макет с колонками равных размеров по ширине и высоте.
css значение: display: flex;
.--align-start Добавьте этот класс, чтобы выровнять flex-элементы по верху оси X
css значение: align-items: flex-start;
.--align-stretch Добавьте этот класс, чтобы выровнять все flex-элементы по одной высоте
css значение: align-items: stretch;
.--align-center Добавьте этот класс, чтобы выровнять flex-элементы по сентру оси X
css значение: align-items: center;
.--align-end Добавьте этот класс, чтобы выровнять flex-элементы по низу оси X
css значение: align-items: flex-end;
.--just-start Добавьте этот класс, чтобы выровнять flex-элементы с лева по оси Y
css значение: justify-content: flex-start;
.--just-center Добавьте этот класс, чтобы выровнять flex-элементы по сентру оси Y
css значение: justify-content: center;
.--just-space Добавьте этот класс, чтобы выровнять flex-элементы одинаково оси Y
css значение: justify-content: space-between;
.--just-end Добавьте этот класс, чтобы выровнять flex-элементы по низу оси Y
css значение: justify-content: flex-end;
.--direction-column Добавьте этот класс, чтобы выстроить колонки
css значение: flex-direction: column;
